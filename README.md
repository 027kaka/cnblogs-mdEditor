# cnblogs md editor
-----

由于博客园的narkdown编辑器不是很好用，撸了一个chrome插件，改善一下用户体验

## feature

- 支持设置主题(light/dark)
- 支持设置字体大小
- 支持生成toc目录🆕
- 支持编辑器全屏

## 安装

- 在浏览器中输入`chrome://extensions`
- 打开开发者模式
- clone仓库
- 点击加载已解压的扩展程序,选择clone下来的app文件夹
- 如果一切正常，打开新增随笔时即可看到变化

## 设置

目前支持设置主题（light 和 dark）
